## md-edit
markdown 文本编辑器（基于 Node.js + Electron）

### 运行

```bash
npm install
npm run dev
```

### 打包为 Windows 可执行程序

#### 1. 准备图标文件（可选）

将应用程序图标文件 `icon.ico` 放在 `build/` 目录下。如果没有图标文件，打包仍可进行，但会显示警告。

#### 2. 执行打包

```bash
# 打包为安装程序（推荐）
npm run build

# 或者打包为目录（用于测试）
npm run build:dir
```

打包完成后，安装程序将生成在 `dist/` 目录下。

#### 3. 安装程序功能

安装程序会自动配置以下功能：

- **文件关联**：将 `.md` 文件关联到 md-edit
- **右键菜单**：
  - **使用 md-edit 打开**：在任何文件或 `.md` 文件上右键，选择"使用 md-edit 打开"
  - **新建 Markdown 文件**：在文件夹或文件夹背景上右键，选择"新建 Markdown 文件"

#### 4. 使用右键菜单

安装后，你可以：

1. **打开 Markdown 文件**：
   - 双击 `.md` 文件（自动关联）
   - 或右键点击任何文件，选择"使用 md-edit 打开"

2. **新建 Markdown 文件**：
   - 在文件夹空白处右键，选择"新建 Markdown 文件"
   - 或在文件夹上右键，选择"新建 Markdown 文件"

### 功能概览

- **Markdown 编辑**：左侧编辑区 + 右侧实时预览
- **菜单栏**：文件 / 编辑 / 段落 / 格式 / 视图 / 主题 / 帮助（参考 Typora）
- **格式操作**：标题、列表、任务列表、加粗、斜体、行内代码、代码块等
- **主题**：Github 与 Night 主题切换
- **Windows 集成**：文件关联、右键菜单支持

你可以在此基础上继续扩展数学工具、表格、图片插入等高级功能。
